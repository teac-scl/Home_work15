JavaScript програма для перевірки успішності студентів з груп GROUP1, GROUP2, GROUP3 з різними оцінками (значення рандомні) та виводуння інформації по рейтингам на екран
Оцінки:
Хороші студенти: 89-100
Середні студенти: 65-88
Слабкі студенти: 0-64


    Вхідні дані:
  • Масиви group1, group2, group3 з даними про студентів (об'єкт спочатку містить два ключа name та gnd (цей ключ для того, щоб задати ім'я файлу з аватаркою: шаблон: якщо gnd="female" то шаблон "w#.png", інакше якщо gnd="male", то шаблон "m#.png", # - рандомне число від 1 до 7 (всього по 7 файлів із кожним видом зображення). Всі інші ключі: група (параметр функції), ID, дисципліна (вибирається із масиву за рандомним індексом елемента), рандомна оцінка, фото додається за допомогою функції editArrKey
  • Дві глобальні змінні averagePoint = 65 (середні результати),hightPoint = 89 (найкращі результати)
